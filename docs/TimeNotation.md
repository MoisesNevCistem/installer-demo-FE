# **Notaciones de Tiempo**

Esta sección es creada con el propósito de explicar la forma correcta para definir las unidades de tiempo que puede observar en la configuración de variable de entorno. 

Explicación de notación:

| Unidad | Descripción                  |
| ------ | ---------------------------- |
| `s`    | Define el tiempo en segundos |
| `m`    | Define el tiempo en minutos  |
| `h`    | Define el tiempo en horas    |
| `d`    | Define el tiempo en días     |
| `w`    | Define el tiempo en semanas  |
| `M`    | Define el tiempo en meses    |
| `y`    | Define el tiempo en años     |
